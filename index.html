🚀 Quick Start

Up and running in seconds

No install needed. Just run with npx.

1

Auto-detect & generate

Scans your project, detects frameworks, generates rules.

$ npx cursorrules init
2

Add specific presets

Manually add rules for any framework. Smart merging, no duplicates.

$ npx cursorrules add react
$ npx cursorrules add tailwind
3

Browse all presets

See every available preset with descriptions and aliases.

$ npx cursorrules list
